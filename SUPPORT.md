# Support

Use GitHub issues for bugs, feature requests, and release questions:

https://github.com/xiaoxiao27110/taproot/issues

When reporting a bug, include:

- Taproot version
- Python version
- VS Code version, if the extension is involved
- Operating system
- Sanitized `nodes.yaml` structure
- The command or MCP tool call that failed
- Full error output with hostnames, usernames, passwords, keys, and tokens removed

Do not paste real `nodes.yaml` secrets, SSH private keys, `.taproot/history.jsonl`, or approval records into issues.
